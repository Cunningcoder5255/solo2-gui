### Solo2 GUI
Solo2 GUI is a GUI frontent for the solo2 command line application written in rust. Solo2 GUI is primarily intended to provide easy access to TOTP OATH functions, which are not easily accessible through other applications like FIDO. In the future, Solo2 GUI will also implement other solo2 command line functions like FIDO and key updating.

# Installation
Simply download the appropriate file from the tagged released page on the right side of github.
